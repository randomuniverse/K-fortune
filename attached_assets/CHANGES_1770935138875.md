## 주의사항
- saju-additions.ts는 독립 파일이 아닌 기존 saju.ts에 **append** 해야 함
- STEM_ELEMENTS, FIVE_ELEMENTS, FIVE_ELEMENTS_HANJA, calculateYearPillar, calculateMonthPillar 등은 기존 saju.ts에 이미 존재하는 변수/함수이므로 중복 선언하면 안 됨
- 만약 FIVE_ELEMENTS / FIVE_ELEMENTS_HANJA가 saju.ts에 없다면 아래를 추가:
  ```typescript
  const FIVE_ELEMENTS = ["목", "화", "토", "금", "수"] as const;
  const FIVE_ELEMENTS_HANJA = ["木", "火", "土", "金", "水"] as const;
  ```
