개발자님의 피드백을 반영하여 `server/fortune-engine.ts`의 `generateYearlyFortune` 함수를 긴급 업그레이드해야 해.
현재 2026년 총평이 너무 짧고, '대운(Major Cycle)' 같은 중요한 이벤트를 놓치고 있어.

**[수정 목표]**
1. **총평(overallSummary) 대폭 확장:** 단순 요약이 아니라, **대운과 세운의 상호작용**을 포함한 깊이 있는 운세 풀이로 변경.
2. **대운(Daewoon) 로직 추가:** 사용자의 대운 정보를 프롬프트에 명시적으로 전달하고, "대운의 교체기"거나 "중요한 대운"일 경우 이를 강력하게 언급하도록 지시.
3. **분량 3배 확대:** 모든 섹션(총평, 사업, 연애, 건강)을 **UI 제약 없이 길게(최소 500자 이상)** 작성하도록 강제.

**[수정할 코드]**
`server/fortune-engine.ts`의 `generateYearlyFortune` 함수를 아래 코드로 **전체 교체**해줘.

```typescript
// [수정] 2026년 연간 운세 생성 (Deep & Rich Version)
export async function generateYearlyFortune(data: {
  name: string;
  year: number;
  saju: any;
  ziwei: any;
  zodiac: any;
}) {
  const individualPrompt = \`
당신은 \${data.year}년(병오년, 붉은 말의 해)의 운세를 분석하는 '정통 명리학 대가'이자 '인생 전략가'입니다.
AI 같은 기계적인 요약투를 버리고, 실제 철학관에서 상담하듯 **깊이 있고 풍성한 통찰**을 제공하세요.

**[치명적 지시사항 - 위반 시 실패]**
1. **분량 강제:** 모든 섹션은 **최소 10문장(500자) 이상** 작성하세요. UI 박스 크기는 신경 쓰지 말고, 할 말을 끝까지 다 하세요.
2. **대운(Big Cycle) 필수 분석:** 제공된 '대운(Daewoon)' 데이터를 반드시 확인하세요. 대운이 바뀌는 시기거나, 현재 대운이 올해와 합/충이 되는 경우 이를 **"인생의 거대한 기회/위기"**로 강력하게 서술하세요. (예: "올해는 10년짜리 대운이 들어오는 엄청난 해입니다!")
3. **구체적 액션 플랜:** "좋습니다"로 끝내지 말고, "무조건 계약하세요", "북쪽으로 가세요" 같은 **점지(Divination)** 수준의 조언을 하세요.

**[작성 가이드라인]**

1. **overallSummary (${data.year}년 총평 - 가장 중요):**
   - **형식:** 편지글이나 심층 칼럼 형식.
   - **내용:** 병오년(세운)과 사용자 사주(일주/대운)의 화학적 결합을 설명하세요.
   - **대운 분석:** "올해는 당신에게 대운이 작용하여..." 라는 문장이 반드시 포함되어야 합니다. 대운이 주는 기회를 놓치지 말라고 강력히 응원하세요.

2. **businessFortune (사업/재물운):**
   - **길게 쓰세요:** 월별 흐름을 요약하지 말고, 올해 전체의 **재물 그릇**을 설명하세요.
   - **비즈니스 모델:** "유통보다는 제조", "확장보다는 내실" 등 구체적 전략 제시.

3. **loveFortune & healthFortune:**
   - 3줄 요약 금지. 상황별(싱글/커플, 신체부위별)로 **상세하게** 서술하세요.

4. **monthlyFlow (월별 흐름):**
   - 기존 방식 유지하되, **"O월은 OO(간지)의 기운입니다"** 포맷 필수로 지킬 것.

**[출력 형식 (JSON)]**
{
  "overallSummary": "대운과 세운을 아우르는 1000자 분량의 심층 총평 (응원과 확신)",
  "businessFortune": "500자 이상의 구체적 사업/재물 전략",
  "loveFortune": "500자 이상의 상세 연애/인간관계 조언",
  "healthFortune": "500자 이상의 건강 관리법 (식이/운동 포함)",
  "keywords": ["키워드1", "키워드2", "키워드3", "키워드4", "키워드5"],
  "monthlyFlow": [
    {"month": 1, "score": 85, "keyword": "문서운", "summary": "1월은 ... (3문장 이상)"},
    ...
  ]
}
\`;

  // 사용자 데이터 프롬프트 구성 (대운 정보 강조)
  const daeunInfo = data.saju.daeun 
    ? data.saju.daeun.slice(0, 3).map((d: any) => \`\${d.age}세 대운: \${d.stem}\${d.branch}(\${d.stemHanja}\${d.branchHanja})\`).join(", ")
    : "대운 데이터 없음";

  const userPrompt = \`
[사용자: \${data.name}] - \${data.year}년 운세 정밀 분석

1. [사주 핵심]
   - 본원(일간): \${data.saju.dayMaster}
   - 일주: \${data.saju.dayPillar?.stemHanja}\${data.saju.dayPillar?.branchHanja}
   - **대운 흐름(중요): \${daeunInfo}** (현재 나이와 대운을 대조하여 대운의 영향을 최우선으로 분석할 것)
   - 용신: \${data.saju.yongShin.element}

2. [자미두수] 명궁 주성: \${data.ziwei.stars.life.map((s: any) => s.name).join(", ")}
3. [별자리] \${data.zodiac.sign}

위 데이터를 바탕으로, 특히 **'대운'의 작용**을 강조하여 \${data.year}년 운세를 아주 상세하게(섹션당 500자 이상) 서술하세요.
\`;

  try {
    console.log(\`[Yearly] \${data.year}년 운세 생성 시작 (Deep Mode)...\`);

    // 3회 반복 생성 (풍성함을 위해 유지)
    const generateOne = async (runIndex: number) => {
      const response = await openai.chat.completions.create({
        model: "gpt-4o",
        messages: [
          { role: "system", content: individualPrompt },
          { role: "user", content: userPrompt },
        ],
        response_format: { type: "json_object" },
        temperature: 0.85 + (runIndex * 0.05), // 회차별로 창의성 높임
      });
      return JSON.parse(response.choices[0].message.content || "{}");
    };

    const [report1, report2, report3] = await Promise.all([
      generateOne(0),
      generateOne(1),
      generateOne(2),
    ]);

    // 종합 프롬프트 (길이 유지 강조)
    const synthesisPrompt = \`
당신은 최종 검증관입니다. 3개의 분석 리포트를 종합하세요.

**[종합 원칙]**
1. **절대 요약 금지:** 3개 리포트의 내용을 합쳐서 **가장 길고 자세한 버전**을 만드세요. 줄이지 마세요.
2. **대운 강조:** "올해는 대운이 들어오는 해입니다" 같은 강력한 멘트를 overallSummary 맨 앞에 배치하세요.
3. **분량:**
   - overallSummary: 1000자 내외
   - business/love/health: 각 500자 내외
   - monthlyFlow: 각 월별 3문장 이상

**[입력 데이터]**
A: \${JSON.stringify(report1)}
B: \${JSON.stringify(report2)}
C: \${JSON.stringify(report3)}
\`;

    const synthesisResponse = await openai.chat.completions.create({
      model: "gpt-4o",
      messages: [
        { role: "system", content: "당신은 요약을 싫어하는 운세 에디터입니다. 내용을 최대한 길고 풍성하게 작성하세요." },
        { role: "user", content: synthesisPrompt },
      ],
      response_format: { type: "json_object" },
      temperature: 0.4,
    });

    const finalResult = JSON.parse(synthesisResponse.choices[0].message.content || "{}");
    
    // 안전장치
    if (!finalResult.monthlyFlow) finalResult.monthlyFlow = report1.monthlyFlow;
    if (!finalResult.keywords) finalResult.keywords = report1.keywords;

    return finalResult;

  } catch (e) {
    console.error("Yearly Fortune Generation Error:", e);
    // 에러 발생 시 기본값 반환
    return {
      overallSummary: "운세 데이터를 분석하는 중 문제가 발생했습니다.",
      businessFortune: "잠시 후 다시 시도해주세요.",
      loveFortune: "잠시 후 다시 시도해주세요.",
      healthFortune: "잠시 후 다시 시도해주세요.",
      keywords: ["분석", "대기"],
      monthlyFlow: []
    };
  }
}
